# NNUE 注釈 A/B 計測サマリ（2025-10-29）

本レポートは https://github.com/SH11235/shogi/pull/195 での対応の P1.5（Near‑final 狭窓最終確認）ON/OFF のA/Bを要約したものです。

## 実験条件（共通）
- 教師: Enhanced、WDL=600、jobs=16、parted出力（`out.part-0001.jsonl`）
- P1（近締切ゲート/アスピ安定化/縮退）は常にON。P1.5のみトグル。
- P1.5 ON 条件: `SHOGI_ZERO_WINDOW_FINALIZE_NEAR_DEADLINE=1`、Δはケース別（2〜3cp）

## 実験1: 1k（TIME=300ms, MPV=3）
- OFF → ON（Δ=1cp, 既定）

- 所見: 短TC/浅深では恩恵小、既定OFFが妥当。
